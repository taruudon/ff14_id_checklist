# FF14 ID装備チェックリスト

FF14（ファイナルファンタジーXIV）のID装備コレクション用チェックリストです。

- パッチ3.0〜7.4の4人用ダンジョンを収録
- ジョブカテゴリ別（ディフェンダー・ヒーラー・ストライカーなど）に切替可能
- チェックデータは各自のブラウザに自動保存されます

## 使い方

https://taruudon.github.io/ff14_id_checklist/
URLをブラウザで開くだけで使えます。登録不要・無料。
